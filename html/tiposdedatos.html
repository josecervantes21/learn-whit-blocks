Bloque de tipos de datos

Texto:

Recuerda que las expresiones son solo valores combinados con operadores, y siempre evalúan hasta un valor único. Un tipo de datos es una categoría para valores, y cada valor pertenece exactamente a un tipo de datos. Los tipos de datos comunes en Python se enumeran en la Tabla 1-2. Los valores **-2** y **30**, por ejemplo, se dice que son valores enteros. El tipo de datos entero (o int) indica valores que son números enteros. Números con punto decimal, como **3.1416**, se denominan números de coma **float(o flotantes).
** Tenga en cuenta que aunque el valor 42 es un número entero, el valor **42.0** sería un número de punto flotante.
Los programas de Python también pueden tener valores de texto llamados cadenas, o **strs** (pronunciado "revuelve"). Rodee siempre su cadena con caracteres de comillas simples (') (como en **'Hola'** o **'¡Adiós mundo cruel!'**) para que Python sepa dónde está la cuerda comienza y termina. Incluso puede tener una cadena sin caracteres '', llamado una cadena en blanco.
El nombre de la función es type. La expresión entre paréntesis se llama argumento de la función. El argumento es un valor o variable que estamos pasando a la función como entrada a la función. El resultado, para la función de tipo, es el tipo del argumento. Es común decir que una función "toma" un argumento y "devuelve" un resultado. El resultado se denomina valor de retorno.
